Разработано с использованием Microsoft Visual Studio 2019.
Версия программы: 1.0 
Системные требования: 
ОС Windows: 7,8,8.1,10. 
Процессор: 1.5GHz 
Оперативная память: 512MB ОЗУ 
Видеокарта: 512MB 
Место на диске: 500МВ 
Необходимые программы и компоненты:
.NET Framework 4.7.2
Microsoft SQL Server 2016 LocalDB Версия 13.1.4001.0 и более поздние версии.

Для работы необходимо скопировать папку с приложением в место хранения файлов приложений или в удобное для вас место и сделайть ярлык для исполняемого файла. 
Рекомендуеться: скопируйте папку и разместите её по следующему пути C:\Program Files, после этого скопируйте файла Hotel_.exe и вставьте его как ярлык на рабочий стол или в любое удобное для вас место.

Приятного пользования!
